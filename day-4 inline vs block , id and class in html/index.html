definations

INLINE: Inline Elements don't start on a new line. It only takes the width required to cover the content.

  • span: A generic inline container for text
  • a: Defines a hyperlink
  • strong: Defines important text
  • input: Defines an input control
  • em: Defines emphasized text
  • img: Embeds an image

block level: it consume whole line and next element start in new line

  • Characteristics of Block-level Elements:

  • Always start on a new line.
  • Take up the full width available.
  • Width and height can be controlled via CSS.
  • Can contain other block-level as well as inline elements.
