Text formatting tags

    < p>: Paragraph.
    < h1>, < h2>, < h3>, < h4>, < h5>, < h6>: Headings
    < strong>: Strong emphasis (typically bold).
    < em>: Emphasis (typically italic).
    < br>: Line break.
    < hr>: Horizontal rule.
